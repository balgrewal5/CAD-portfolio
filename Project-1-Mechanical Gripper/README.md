# Month 1 — Mechanical Gripper

## Project Overview
A geared two-finger mechanical gripper designed in Autodesk Fusion 360 to develop core CAD, assembly, technical drawing, and FEA skills.

---

## Skills Covered
- Parametric CAD modelling
- Fully constrained sketching
- Assemblies and joints
- Spur gear integration
- Technical drawings
- Material assignment
- Static stress FEA
- Stress concentration analysis
- Geometry optimisation

---

## Software Used
- Autodesk Fusion 360

---

## Project Deliverables
- 3D CAD assembly
- Engineering drawings
- Parts list (BOM)
- Material selection
- FEA stress analysis
- Safety factor analysis
- Design iteration with fillets

---

## FEA Summary
A static stress analysis was carried out on the gripper jaw using a 20 N load to evaluate stress distribution, displacement, and safety factor. Geometry improvements were later introduced to reduce stress concentrations around the pivot region.

---

## Key Learning Outcomes
- Understanding stress flow through geometry
- Learning how fillets reduce stress concentration
- Developing engineering drawing workflow
- Building mechanically realistic assemblies
